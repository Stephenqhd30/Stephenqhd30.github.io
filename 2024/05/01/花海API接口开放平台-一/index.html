数据库表的设计

接口信息表 （interface_info）

  • id 用户的id
  • name 接口名称
  • description 接口描述
  • url 接口地址
  • type 请求类型
  • requestHeader 请求头
  • responseHeader 响应头
  • status 接口状态 （0 - 关闭 1 - 开启）
  • isDelete 是否删除 （0 - 正常 1 - 删除）
  • createTime 创建时间
  • updateTime 更新时间

项目的脚手架

  • 前端使用 ant desgin pro 框架自动生成代码
  • 后端使用 springboot init 初始化代码

基础功能

  • 后端快速开发增删改查的代码
  • 前端通过 OpenAPI + Swagger 快速生成代码
